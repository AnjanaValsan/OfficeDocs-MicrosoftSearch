# FAQs

These are the frequently asked questions that organizations and users have about administering and using Microsoft Search in Bing.
  
## What's the value proposition for Microsoft Search in Bing?

Microsoft Search in Bing is a secure way to search both your work and web content. This kind of integration across web and enterprise is only available with Microsoft. For compliance-related information, see [Security for Microsoft Search in Bing.](overview/security.md)
  
## What Microsoft Search in Bing features are available now?

For a complete list, see [Features of Microsoft Search in Bing.](overview/features.md)
  
## Does Microsoft Search in Bing support advanced query understanding?

Yes, Microsoft Search in Bing parses query intent from larger phrases. This feature uses AI to learn common superfluous phrases users add to their queries that don't impact their search intent. For example, when a user searches for 'tell me more about how to change my password please' we extract the less important words from the query and trigger based on the relevant ones like 'change password.'
  
This feature will not override keywords set in the Admin portal.
  
## Does Microsoft Search in Bing search for files on-premises as well as the cloud?

Microsoft Search in Bing supports searching for documents that live on SharePoint Online, OneDrive for Business, as well as over hybrid on-premises SharePoint and SharePoint Online for the most common Office file types. If you can search on-premises content in SharePoint Online, you should be able to find on-premises content using Microsoft Search in Bing. 
  
## Does Microsoft Search in Bing replace other enterprise search experiences?

Microsoft Search in Bing is a complementary offering that brings together results from multiple sources. For more information about supported file types and sources, see the next FAQ.
  
## What file types and sources does Microsoft Search in Bing support?

We support the following content sources:
  
- SharePoint Online
    
- Hybrid SharePoint (on-premises + SPO)
    
- OneDrive for Business
    
The following file types are surfaced in file search and appear on the Files tab for People and Groups:
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF

## What are the content sources for the people card?

The people card derives information from Azure Active Directory, Exchange Online, and SharePoint Online.
  
## Do Microsoft Search in Bing users earn Microsoft Rewards?

Microsoft Search in Bing searches do not earn Rewards points and are never associated with a user's Microsoft account.
  
## Does Microsoft Search in Bing respect existing file permissions?

Microsoft Search in Bing respects security trimming from the source. Users can only see information they have access to.

## How does Microsoft Search in Bing use Azure Active Directory?

Microsoft Search in Bing uses Azure Active Directory to authenticate and authorize access to your company's data. Microsoft Search in Bing conforms to the identity handling practices standard across all Microsoft products. This means your users may be automatically signed in through single sign-on.

## Why am I signed into Bing automatically?

Microsoft Search in Bing uses AAD single sign-on to automatically sign you in with your work or school account when you've signed in to an Office 365 app, or Windows 10. You can sign out at any time.
  
## What is Bing for business?

## How does Microsoft Search in Bing order result cards in the All results carousel?

Microsoft Search in Bing uses intelligent ranking algorithms to order result cards based on relevance.
